### Case 1. 로그인 메일 주소를 잊어버린 경우

응대 플로우

![계정문의안내프로세스-메일주소 모름.png](attachment:81c18336-51a2-4302-91f4-1d8b0ecf3dc0:계정문의안내프로세스-메일주소_모름.png)

1. 고객에게 이용하고 있는 채널명 확인 요청, 팔로업 알림에 입력된 전화번호 등으로 어드민에서 정보 검색
2. 예상되는 이메일 주소를 ch*****@channel.io 형태로 암호화하여 공유
    - 고객이 위 정보로도 메일주소를 알지 못하는 경우라면, 본인확인이 불가능 하기에 재가입 해야함 (#계정찾기_채널초대및권한변경신청)
    - 이 때, 발생하는 시트 비용은 1회에 한하여 제공 가능

### Case 2. 비밀번호를 잊어버린 경우

1. 직접 비밀번호를 재설정하는 방법 전달 
    - https://desk.channel.io/#/password_reset (#계정_비밀번호_재설정링크)
2. 비밀번호 변경 링크가 전달되는 메일을 못받는 경우, [Case 4 메일 수신 불가 상황 플로우](https://www.notion.so/HT_008-1f174b55ec7c803ab3abcfa9b8239f81?pvs=21)에 따라 응대 진행

### Case 3. 인증번호를 못받고 있는 경우

(주의) 어떤 경우라도 인증번호를 임의로 전달하는 것은 불가능

1. 인증번호가 전달되는 메일을 못받는 경우, [Case 4 메일 수신 불가 상황 플로우](https://www.notion.so/HT_008-1f174b55ec7c803ab3abcfa9b8239f81?pvs=21)에 따라 응대 진행

### Case 4. 비밀번호 변경, 인증번호 메일이 안오는 경우

메일이 안오는 원인

- 채널톡에서 전송하는 메일이 스팸차단 된 경우 ([4-1](https://www.notion.so/HT_008-1f174b55ec7c803ab3abcfa9b8239f81?pvs=21))
- 가입 시 오타로 인해 메일 주소를 잘못 입력한 경우 ([4-2](https://www.notion.so/HT_008-1f174b55ec7c803ab3abcfa9b8239f81?pvs=21))
- 사이트의 도메인이 변경된 경우 ([4-2](https://www.notion.so/HT_008-1f174b55ec7c803ab3abcfa9b8239f81?pvs=21))
    - 예시) inus@zoyi.co를 사용하다가 inus@channel.io로 도메인이 변경됨, 기존 zoyi.co 메일 확인 불가능
- 다른사람의 메일에 접속하는 경우 ([4-2](https://www.notion.so/HT_008-1f174b55ec7c803ab3abcfa9b8239f81?pvs=21))
    - 공용 계정으로 가입한 계정에 메일을 확인할 수 없는 경우 (예시 feedback@channel.io)
    - 퇴사자의 메일 계정에 접속해야 하는 경우

**4-1. 이메일 주소(계정)에 접근 가능한 경우**

1. 메일 차단 해제 시도하기 (#계정찾기_차단해제_내부용)
    1. 고객에게 내게 쓴 메일함, 스팸함 등 전체 메일함 확인 요청 및 차단 메일 확인 요청
    2. 사용하는 이메일 서비스 확인 후, #스팸제외설정방법 검색하여 스팸 제외 요청
    3. 계정 이메일 주소 시스템(채널톡 전송 시스템) 차단 상태 확인 요청(메인: 브라운, 서브: 딜런)
    4. (끝까지 해결 안되면) 앤써니, 알렉스, 조조 초대하여 논의 해보기
2. (비밀번호 변경 링크만) 고객이 이메일 계정으로 인증 메일을 보내면 이를 확인 후 메일 주소로 비밀번호 재설정 링크 전달 (#계정찾기_비밀번호재설정링크) 
    - 이때 이메일 회신은 반드시 계정 주소로 직접 발송(love@ 1ove@ 등으로 조작할 수 있으므로)
    - 인증번호는 절대 전송 불가
3. 끝까지 메일을 받을 수 없다면 계정을 새로 만들어야 함 (#계정찾기_채널초대및권한변경신청) 
    - 이 때, 발생하는 시트 비용은 1회에 한하여 제공 가능

**4-2. 이메일 주소(계정)에 접근 불가한 경우**

1. 본인 계정 여부 확인 불가하므로 계정을 새로 만들어야 함 (#계정찾기_채널초대및권한변경신청)
