## 목적

1. 푸시 알림이 오지 않는 문제 해결

## 주의사항

1. 고객이 예민하게 반응하거나, 빠른 답변을 필요로 하는 경우가 많으니 빠르고 정확한 답변 중요
2. 고객 상황 확인 후 기본 설정이 안된 경우 해당 내용 먼저 안내
3. 정확한 상황 파악을 위해 체크 필요한 내용이 많으니 잘 정리하여 전달 필요 (매크로 활용하기)

## 문제 해결 프로세스

### 문제 상황 확인

**`매니저 확인 필요 사항`** 

- 어드민에서 푸시 알림 설정 확인
    
    ![image.png](https://prod-files-secure.s3.us-west-2.amazonaws.com/01a7f0f5-244e-40f0-989f-eb88d0eebfb3/f8cba521-720e-4fd7-a40e-8d2bb12fe9d8/image.png)
    
    - **Operator**
        - operator: 오퍼레이터 시트 ON/OFF
        - receiveUnassignedChatAlert: 담당자 없는 채팅 알림 ON/OFF
        - operatorEmailReminder: 이메일 리마인더
        - receiveUnassignedMeetAlert: 담당자 없는 전화 알림 ON/OFF
    - **Channel settings**
        - defaultGroupWatch: 팀챗 알림 - 그룹
        - defaultDirectChatWatch: 팀챗 알림 - 다이렉트 메시지
        - defaultUserChatWatch: 수신 알림 - 유저챗
        - doNotDisturb: 방해금지 모드 ON/OFF
        - displayAsChannel: 채널 프로필로 응대 ON/OFF
        - showPrivateMessagePreview: 팀챗 알림 - 비공개 메시지 미리보
        - defaultAllMentionImportant: 소리 및 기타 - 중요 메시지 정의 - 전체 멘션 체크 여부
        - userMessageImportant: 소리 및 기타 - 중요 메시지 정의 - 고객의 메시지 체크 여부
        - autoAssignCapacity: 최대 자동 배정
        - chatAlertSound: 소리 및 기타 - 메시지 소리

**`고객 확인 필요 사항`**

- 어떤 푸시 알림이 오지 않는지
    - 팀챗/수신함에 대한 모든 팝업/소리 알림 모두 오지 않는지
    - 신규 상담에 대한 알림이 오지 않는지
    - 담당자로 배정된 상담 알림이 오지 않는지
    - 채팅 / 미트 링이 뜨지 않는지
- 현재 설정 및 사용 환경 확인
    1. PC 앱 / 브라우저 / 모바일 앱 중 이용 형태가 무엇인지
        1. 지금 PC앱으로 접속중이면, 다른 경로(브라우저, 모바일)에서도 알림이 오지 않는지
    2. 푸시 알림 설정 ON + 방해 금지 모드 OFF  → [어드민 확인 방법](https://www.notion.so/f3fcca57230a45e1b2c33b4d89667bee?pvs=21) / [해결 방법](https://www.notion.so/f3fcca57230a45e1b2c33b4d89667bee?pvs=21)
    3. 기기(PC/모바일) 자체 채널톡에 대한 알림 설정 ON + 방해 금지/집중 모드 OFF → [해결 방법](https://www.notion.so/f3fcca57230a45e1b2c33b4d89667bee?pvs=21)
    4. 하나의 계정을 여러 명이 사용하고 있는지 
    5. 모든 매니저에게 푸시 알림이 오지 않는지
    6. 워크플로우에 팀/담당자 배정 액션이 추가되어 있는지 → [해결 방법](https://www.notion.so/f3fcca57230a45e1b2c33b4d89667bee?pvs=21)
        1. 배정되어 있는 팀이 상담 자동 배정 ON 상태인지

### 문제 원인 파악 및 해결책 안내:

`케이스 1`  푸시 알림 설정이 되어 있지 않거나, 방해금지 모드 ON 일 때

- 푸시 알림 설정 안내 ([가이드](https://docs.channel.io/help/ko/articles/3fccba8f-%ED%91%B8%EC%8B%9C%EC%95%8C%EB%A6%BC-%EC%84%A4%EC%A0%95))
    - 알림 옵션: 모든 알림 받기
    - 담당자 없는 채팅/전화 알림 ON
    - 방해금지 OFF (멀티 채널 운영중인 경우 [멀티 채널 응대 사용중] 확인하기)

`케이스 3` 1개의 계정을 여러 매니저가 동시에 사용중일 때

- 동시 사용 시, 다른 PC에서 알림을 확인하면, 나머지 매니저에게는 알림이 오지 않을 수 있기 때문에 권장하지 않음
- 매니저 각각 계정 생성하여 사용을 권장

`케이스 4-1`  워크플로우에 팀 배정 / 담당자 배정 액션이 있을 때

- 배정된 팀 / 담당자에게만 알림이 감
    - 해당 매니저가 팀에 속해있거나, 담당자인지 확인하기
- 팀에서 상담 자동 배정 기능을 쓰고 있다면 자동 배정된 담당자에게만 알림이 감
    - 본인이 담당자로 배정되어 있는지 확인

`케이스 4-2`  워크플로우에 팀 배정 / 담당자 배정 액션이 없고, 미트 링이 오지 않을 때

- 전화번호에서 특정 팀에 '라우팅' 설정이 되어 있으면, 해당 번호로 전화 유입 시 라우팅 된 팀원에게만 링이 뜸
    - 라우팅 된 팀에 속해있는지 확인
    - 라우팅 없는 번호로 전화 걸면 링이 뜨는지 확인
    
    ![image.png](https://prod-files-secure.s3.us-west-2.amazonaws.com/01a7f0f5-244e-40f0-989f-eb88d0eebfb3/1a48951a-e60d-4a75-a39b-7986907912e2/image.png)

`케이스 5` 케이스 1~4번 모두 해당되지 않을 때

- 재부팅 시에도 동작 안되는지 확인 (1→4 순서로)
    1. 새로고침
    2. 브라우저/앱 종료 후 재실행 
    3. 다른 브라우저로 접속 / 앱 재설치
    4. PC 종료 후 재실행

`케이스 6` 케이스 5번으로 해결되지 않고, 특정 고객사에게만 푸시 알림이 오지 않을 때

- 개발팀 초대

- 에스더 정리
    - ✅ **1차 해결 방법**
        - 모든 알림 받기

        - 담당자 없는 전화/채팅 알림 ON
        - 미트 알림 받을 장치 체크
        - 방해금지 OFF
            - [채널 프로필 → 멀티 채널 응대]에서 채널별 설정 한 번에 확인 가능
        
        ![image.png](https://prod-files-secure.s3.us-west-2.amazonaws.com/01a7f0f5-244e-40f0-989f-eb88d0eebfb3/54b71ba8-c1c0-4252-b5c3-c6c52c9a8195/image.png)
        
        ![image.png](https://prod-files-secure.s3.us-west-2.amazonaws.com/01a7f0f5-244e-40f0-989f-eb88d0eebfb3/e8ef3a63-c877-41ae-b506-09c73ee9007f/image.png)
        
    - 설정이 되어 있지 않은 경우 → 최대한 어드민에서 확인해서 안내하기
        - 전체 알림이 오지 않는 경우
            - 방해금지 OFF면 모든 푸시 알림이 OFF됨
            - 푸시 알림 설정 잘 되어 있으나, 팝업 알림이 뜨지 않으면, 기기 자체 알림 설정 확인
                - 채널톡 앱 → 기기에서 채널톡 앱 알림ON 상태인지 확인
                - 웹 브라우저 → 기기에서 브라우저 알림 ON 상태인지 확인
                - 방해금지/집중모드 상태인지 체크
        - 수신함 알림이 오지 않는 경우
            - 신규 상담이 오지 않는다면
                - ‘담당자 없는 채팅/전화’ 알림이 ON 상태 확인
            - 담당자가 있는 상담에 대한 알림이 오지 않는다면
                - 담당자/팔로워로 참여중인 상태인지 확인
