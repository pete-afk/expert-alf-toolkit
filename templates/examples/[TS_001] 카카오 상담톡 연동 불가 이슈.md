## 주의사항

## 문제 해결 프로세스

### 문제 상황 확인

`매니저 확인 필요 사항`

고객사 어드민 접속하여 해당 채널과 카카오 상담톡 연동이 제대로 안된 게 맞는지 확인 

- 확인 방법
    - [채널] → [App] → [App Kakao]에서 확인 가능
    
    ![image.png](https://prod-files-secure.s3.us-west-2.amazonaws.com/01a7f0f5-244e-40f0-989f-eb88d0eebfb3/62fcff2b-026a-4faa-b9ca-d4ccd289f880/image.png)
    

`고객 확인 필요 사항`

고객에게 어떤 오류 메시지를 받았는지 문구 및 캡쳐 이미지 요청

### 해결책 안내

`오류 메시지에 따른 분류`

- **토큰 인증 대기 중**
    
    ![image.png](https://prod-files-secure.s3.us-west-2.amazonaws.com/01a7f0f5-244e-40f0-989f-eb88d0eebfb3/fbd6fdb6-df32-4158-9728-64d40309d813/image.png)
    
    - 원인:
        - 등록된 *관리자 휴대폰 번호로 인증번호 카톡이 전송되는데, 해당 인증번호가 입력되지 않았기 때문
    - 해결책:
        - 관리자 연락처로 도착한 인증번호 입력 필요
            
            *이 때, 관리자 휴대폰 번호는 [[카카오 채널 관리자센터](https://center-pf.kakao.com/profiles) → 관리 → 관리자]에 등록된 담당자 번호
            
        - 관리자로 등록되어 있는데도 해당 연락처로 인증번호가 안 오는 경우:
            - [개인 휴대폰 카카오톡 → 친구 → 우측 상단 톱니바퀴 → 친구 관리 → 차단 친구 관리]에서 **카카오톡 비즈메세지** 채널이 차단되어있다면 차단 해제 필요
            - [[카카오 채널 관리자 센터](https://center-pf.kakao.com/)] → [관리] → [관리자]에서 다른 사람을 관리자로 등록하고 해당 번호로 오는 인증번호 입력
        
        ![image.png](https://prod-files-secure.s3.us-west-2.amazonaws.com/01a7f0f5-244e-40f0-989f-eb88d0eebfb3/6c75c073-546a-426a-90a5-e66c68bf9e4b/image.png)
        
- **카카오 api 요청에 실패했습니다. Reason: 등록하려는 발신프로필 채널의 고객센터 정보 입력이 필요합니다. 채널관리자센터 > 프로필 > 채널 홈 설정 > 기본정보 > 고객센터 입력 에서 고객센터 연락처 정보를 입력해주세요.**
    
    ![image.png](https://prod-files-secure.s3.us-west-2.amazonaws.com/01a7f0f5-244e-40f0-989f-eb88d0eebfb3/8488a710-5830-4ed4-b4df-be955ed67b00/image.png)
    
    - 원인: 카카오 채널 관리자 센터에서 고객센터 연락처 정보가 입력되지 않았기 때문
    - 해결책:
        - [[카카오 채널 관리자센터](https://center-pf.kakao.com/profiles) → 프로필 → 채널 홈 설정 → 기본정보 → 고객센터 입력]에서 고객센터 연락처 정보 입력
            
            ![image.png](https://prod-files-secure.s3.us-west-2.amazonaws.com/01a7f0f5-244e-40f0-989f-eb88d0eebfb3/6715d139-5f0c-411c-99ff-e0f77868da26/image.png)
            
- **카카오 api 요청에 실패했습니다. Reason: 카카오톡 채널 상태를 확인 해주세요.**
    
    ![image.png](https://prod-files-secure.s3.us-west-2.amazonaws.com/01a7f0f5-244e-40f0-989f-eb88d0eebfb3/62e7cb40-99ef-49db-9e37-e5327632f6a5/image.png)
    
    - 원인:
        - 카카오 비즈니스 채널이 비공개 상태일 때 발생
    - 해결책:
        - [[카카오 채널 관리자 센터](https://center-pf.kakao.com/profiles)] → [프로필] → [프로필 설정] → [옵션 설정] → [공개 설정] → [채널 공개] ON
            
            ![image.png](https://prod-files-secure.s3.us-west-2.amazonaws.com/01a7f0f5-244e-40f0-989f-eb88d0eebfb3/a8d5c3fc-3f94-4de1-923d-c61d13a6a53d/image.png)
            
- **카카오 api 요청에 실패했습니다. Reason: 비즈니스 채널 전환이 필요한 프로필입니다.**
    
    ![image.png](https://prod-files-secure.s3.us-west-2.amazonaws.com/01a7f0f5-244e-40f0-989f-eb88d0eebfb3/5e2b1c07-e249-4141-8bcc-0b8ade9bfb10/image.png)
    
    - 원인:
        - 카카오에서 비즈니스 인증을 받지 않았을 때 발생
    - 해결책:
        - 카카오에서 해당 채널에 대한 비즈니스 인증 진행 필요
        - [[카카오 채널 관리자 센터](https://center-pf.kakao.com/profiles)] → [관리] → [비즈니스 채널 신청] → [신청]
        
        ![image.png](https://prod-files-secure.s3.us-west-2.amazonaws.com/01a7f0f5-244e-40f0-989f-eb88d0eebfb3/600a37a1-ef1c-4204-94b5-887e1db4c443/image.png)
        
- **카카오 api 요청에 실패했습니다. Reason: 요청한 번호가 카카오톡 채널 관리자 알림 설정 되어있는지 확인해주세요.**
    
    ![image.png](https://prod-files-secure.s3.us-west-2.amazonaws.com/01a7f0f5-244e-40f0-989f-eb88d0eebfb3/0cb47ba2-4c91-4ae5-a013-03a278d0146a/image.png)
    
    - 원인: 카카오 비즈니스 채널에 관리자로 등록된 계정의 휴대폰 번호와 일치하지 않을 때 발생
    - 해결책:
        - 해당 카카오 비즈니스 채널에 관리자로 초대되었는 지 확인 후,
            - 관리자로 초대가 되어 있다면, [[카카오 채널 관리자 센터](https://center-pf.kakao.com/profiles)] → 우측 상단 [이메일] 클릭 → [내 정보] 에서 전화번호 수정을 통해 재인증 필요
                
                [내 정보에서 전화번호 수정 방법](https://cf.channel.io/document/spaces/6/articles/76/revisions/223/usermedia/662b0f4d9f1a2a3c5905)
                
            - 관리자로 초대가 안 되어 있다면, 해당 채널의 마스터 또는 매니저가 [[카카오 채널 관리자 센터](https://center-pf.kakao.com/profiles)] → [관리] → [관리자]에서 매니저 초대 가능
                
                ![image.png](https://prod-files-secure.s3.us-west-2.amazonaws.com/01a7f0f5-244e-40f0-989f-eb88d0eebfb3/5dc15744-f521-412f-bb77-cf5dc426dbe3/image.png)
                

- **카카오 api 요청에 실패했습니다. Reason: 3018**

![image.png](https://prod-files-secure.s3.us-west-2.amazonaws.com/01a7f0f5-244e-40f0-989f-eb88d0eebfb3/28e6f624-d121-4a74-9802-502b076a592b/image.png)

카카오 측에서 인증 메시지를 전송할 수 없을 때 발생하는 오류

- 알림톡을 차단한 사용자일 경우 발생 (카카오 비즈니스 채널을 차단한 경우, 인증 메시지가 전송되지 않을 수 있음)
    - [개인 휴대폰 카카오톡 → 친구 → 우측 상단 톱니바퀴 → 친구 관리 → 차단 친구 관리]에서 **카카오톡 비즈메세지** 채널이 차단되어있다면 차단 해제 필요
- [[카카오 채널 관리자 센터](https://center-pf.kakao.com/profiles)] → 우측 상단 [이메일] → [내정보]에 등록된 번호와 동일한 번호인지, 비즈메시지 알림톡을 받을 수 있는 상태가 맞는지 확인 필요
    
    ![image.png](https://prod-files-secure.s3.us-west-2.amazonaws.com/01a7f0f5-244e-40f0-989f-eb88d0eebfb3/1bb4d420-b9db-49ee-9c45-484ed0374861/image.png)
    
- 카카오톡 미사용자 혹은 7일 간 미접속자일 경우 발생 가능
- 카카오톡 하위 버전 사용자(6.0.0 미만)일 경우 → 기기에 다운로드된 카카오톡 버전 업데이트 필요
- **카카오 api 요청에 실패했습니다. Reason: 다른 파트너에 의해 이미 상담톡을 이용중인 프로필입니다. 상담톡 이용을 진행하고자 하는 경우 카카오에 문의해주세요.**
    - 이미 다른 채널톡 채널에 연동되어있는 동일한 카카오 상담톡을 새로운 채널에 연동하려고 했을 때 발생
    - 다른 서비스(루나소프트, 해피톡 등)에 연동이 되었을 때 발생 → 기존 채널 또는 서비스와의 연동 해제 후, 채널톡과 새로 연동 필요
