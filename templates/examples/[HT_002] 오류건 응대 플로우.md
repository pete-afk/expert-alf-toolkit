1. 고객이 불편을 겪고 있는 상황인만큼 빠르게 인사 드리고 응대 시작하기

1️⃣ **이슈 파악**

1. 현재 오류 상황 정확히 파악하기
    - 문제 상황에 대해 고객에게 잘 질문하기 (여러 번 물어보는 거 고객도 귀찮아함)
        - 오류 발생 경로(PC앱 / 브라우저 / 모바일앱)
        - 모든 경로에서 발생하는지
        - 모든 매니저에게 발생하는지
        - 특정 환경에서만 발생하는지
        - 기타 문제 상황에 대한 특이점/공통점
    - 테스트 채널에서 오류 상황 재현 체크
2. 팀챗 버그 리포트((버그방, 각 TF방 등), 관련 문의, 노션/매크로 체크
    - “이전에 문의했었는데~” 로 시작하면 이전 상담 꼭 체크
    - 확인까지 시간 필요 시, 꼭 양해 메시지 보내기
    - 이미 팀에 공유된 이슈라면 기본 조치 방법 안내 후 재현되는지 체크
        - 비파괴적인 순서로 안내하기 (새로고침>종료>삭제)
        - #매크로 잘 활용하기
            - 강력 새로고침 → #강력새로고침 (일렉트론, 브라우저 모두 생성함)
            - 클리어 스토리지 → #클리어스토리지_삭제
            - 앱 재설치 → #일렉트론
    - 참고하면 좋을 이슈 모음 노션
        - [“문자메시지가 안 가요” 체크리스트](https://www.notion.so/1765c65bab2d4d9f8aab162b83417b79?pvs=21)
        - [웹 프론트 에러 구분](https://www.notion.so/e442731e5d274794837454163cf33433?pvs=21)
        - [일렉트론 이슈 케이스 수집(윈도우)](https://www.notion.so/1df6f8e9972e4b989a26e5f9e677417f?pvs=21)
        - [Socket 이슈 대응](https://www.notion.so/Socket-1c2a7a5e33df410f879322655a278d49?pvs=21)
3. 관련 담당자 확인 및 초대 + 상황 전달
    - [누구에게 물어볼까요? 誰に聞いたらいいですか？ Who should I ask?](https://www.notion.so/Who-should-I-ask-2b20827654da408292b6adb7e37f8178?pvs=21)
    - 오류 상황과 함께 제품팀에 함께 전달하면 좋을 내용
        - 이슈 발생 환경(접속 경로/기기, 매니저 등)
        - 고객/유저챗 링크
        - 워크플로우 ID
4. 이슈 관련 고객이 궁금해 할 내용 미리 파악하기
    
    ex. 채널톡 이슈로 통계 집계가 제대로 되지 않은 경우, 이슈 해결되면 잘못된 지표 다시 정상화 되는지
    

2️⃣ **고객 안내**

1. 채널톡 이슈라면 사과 필수
    - 문제 해결도 중요하지만, 고객 소통도 경험을 챙기는 중요한 요소. 잘 챙기자!
2. 오류 상황 안내
    - **[원인 - 영향 받은 부분 - 현재 상태 -개선 계획]** 순서로 안내
    - 업데이트 후 달라진 부분이 있다면, **기존→현재** 순서로 안내
    - 고객은 우리 생각보다 더 더 모른다.  알기 쉽게 설명 중요.
        - 개발자가 아니라면 개발 용어 사용 지양하기
3. 추가 안내가 필요한 경우, 시간 양해 구하기
    - 대략적인 시간이 함께 안내되면 좋음
    - (고객이 불안해하지 않게) “놓치지 않고 안내드리겠다”는 안심멘트 추가

3️⃣ **팀 공유**

1. 채널톡 이슈라면 빠르게 @KR-CX 멘션하여 팀방 공유 (이슈레이징)
2. 세일즈 관리 채널이면 담당자에게 상황 공유(팀챗/내부대화)
    - 어드민 → bizManager 확인
    - 팀방에 채널명 검색해서 관련 담당자 체크

4️⃣ **팔로업**

1. 팔로업 필요 시, 상담 [보류]하기
2. 중간 중간 이슈 해결 상황 체크하기
    - 제품팀도 팔로업 놓칠 수 있음. CX팀이 주기적으로 체크해야 함
    - 관련 팀챗 스레드를 내부대화에 남겨두면 빠른 팔로업 가능
3. 해결 완료 시 안내 드리기
    1. 약속된 시간보다 오래 소요되는 경우, 중간 공유 필요
4. 태그 부착하기
    - 채널톡 이슈면 → Issue
    - 해결 → solved
    - 미해결 → Unresolved
