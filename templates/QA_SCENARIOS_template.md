## QA 실행 가이드

1. 각 시나리오의 "구체적 질문"을 ALF 채팅에 입력
2. **기대 응답 포인트**와 실제 ALF 응답을 비교
3. "추상적 질문"으로도 동일하게 테스트 (의도 인식 확인)
4. Task 시나리오는 실행 결과까지 확인
5. Task/지식 충돌 체크: 정보 문의를 태스크가 가로채지 않는지 확인
6. QA 결과란에 pass/fail + 메모 기록
