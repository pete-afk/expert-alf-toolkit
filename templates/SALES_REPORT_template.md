## 부록

### A. ALF 4-Layer 시간 절감 분석 방법론

1. **완전 해결 (100%)**: 모든 처리 단계를 지식(RAG)/태스크(TASK)/규칙으로 자동 완결할 수 있는 케이스. 상담사 개입 불필요.
2. **승인노드 (~90%)**: ALF가 처리를 완료하고, 상담사가 결과를 확인·승인만 수행하는 케이스. 실질적 처리 시간 최소화.
3. **초벌 상담 (70-80%)**: ALF가 정보 수집·초기 안내를 완료하고, 상담사가 남은 부분을 마무리하는 케이스.
4. **주제 분류 (30-50%)**: ALF가 문의를 자동 분류한 뒤 적합한 상담사에게 연결하는 케이스.
5. **시간 절감 가중평균**: 각 SOP의 레이어별 비율 × 절감률 × 커버리지 건수로 전체 가중평균 산출.

### B. 용어 설명

| 용어 | 설명 |
|------|------|
| HT (How-To) | 정보 제공 및 프로세스 안내 유형 SOP |
| TS (Troubleshooting) | 문제 진단 및 해결 유형 SOP |
| **지식(RAG)** | 문서·FAQ 기반 자동 응답 (별도 API 불필요) |
| **태스크(TASK)** | 외부 시스템 API 호출 (주문 조회, 배송 추적 등) |
| **규칙** | 조건 기반 분기 처리 (보증 기간 체크, 에스컬레이션 등) |
| 완전 해결 | ALF 단독 완결, 상담사 개입 불필요 (시간 절감 100%) |
| 승인노드 | ALF 처리 완료, 상담사 1-click 승인 (시간 절감 ~90%) |
| 초벌 상담 | ALF 정보수집·초기 안내, 상담사 마무리 (시간 절감 70-80%) |
| 주제 분류 | ALF 자동 분류 후 상담사 연결 (시간 절감 30-50%) |
