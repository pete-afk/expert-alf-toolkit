# Agent SOPs

이 디렉토리에는 Agent SOP 파일들이 위치합니다.

## 구조

각 SOP는 `.sop.md` 형식으로 작성되며, Claude Code Skills와 통합됩니다.

## 예정된 SOPs

- `stage1-clustering.sop.md` - Stage 1: 클러스터링 (Python)
- `stage2-extraction.sop.md` - Stage 2: 패턴 추출 (LLM)
- `stage3-sop-generation.sop.md` - Stage 3: SOP 생성 (LLM)
- `excel-to-sop-pipeline.sop.md` - 전체 파이프라인 통합
